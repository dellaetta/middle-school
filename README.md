# Work from middle school
This repository contains a collection of projects I created during middle school, including after-school class assignments and personal projects. 

## Projects
- Dogs: An HTML website about dogs, created to practice basic web design and structure.
- Fortune teller: A Python program inspired by paper fortune tellers that generates responses based on user input.
- Guessing game: A Python program that predicts facts about you by asking a series of questions.
- Stop Watch: A Java program that functions as a 10-second stopwatch.
